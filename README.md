# Student Management System

A Java-based GUI application built using Swing and SQLite.

## Features
- Add Student
- View Students
- Search Student
- Delete Student
- Show Statistics (Average & Top Student)

## Technologies Used
- Java (OOP)
- Swing (GUI)
- SQLite (Database)

## How to Run
1. Compile:
   javac -cp ".;sqlite-jdbc-3.51.3.0.jar" *.java

2. Run:
   java -cp ".;sqlite-jdbc-3.51.3.0.jar" SwingApp
